Inspect uses a small set of open source packages.

  • swift-certificates X.509 certificate parsing and models.
    apple/swift-certificates
  • swift-asn1 ASN.1 encoding used by the certificate layer.
    apple/swift-asn1
  • swift-crypto Cryptographic primitives for tests and tooling.
    apple/swift-crypto
  • tun2proxy Rust TUN proxy for Live Monitor traffic forwarding.
    tun2proxy/tun2proxy
