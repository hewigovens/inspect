Inspect

TLS certificate inspector with passive Live Monitor for iPhone, iPad, and Mac.

Download on the App Store
Live Monitor

Observe TLS traffic
without interception

Certificates are captured from live handshakes and shown in a scrollable feed. Nothing is intercepted or modified.

  • Certificate chains and SNI from live TLS handshakes
  • Detects certificate changes across repeat observations
  • All data stays on-device

Inspection

  • Full certificate chain with X.509 detail
  • Redirect-aware multi-hop inspection
  • OCSP and CRL revocation checking
  • Certificate Transparency SCT log parsing
  • Export certificates as DER or PEM

Security

  • Trust failures and hostname mismatch
  • Expired or not-yet-valid certificates
  • Weak keys and signature algorithms
  • Suspicious CA usage on leaf certificates
  • Interception product detection
  • Certificate change monitoring

Integration

  • Safari extension for inspecting the current page
  • macOS Share extension for URLs from any app
  • iOS action extension from share sheet
  • Home Screen widget for Live Monitor status
  • Open source — Swift 6, SwiftUI, Rust
